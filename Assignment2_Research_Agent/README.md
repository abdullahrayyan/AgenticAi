# 🤖 Autonomous Research Agent (Assignment 2)

## 📌 Project Overview
This project demonstrates an **Autonomous Research Agent** built using the **LangChain** framework. The agent is designed to take a complex topic as input, browse the web and Wikipedia for information, and synthesize a professional, structured report.

By utilizing the **ReAct (Reasoning and Acting)** logic, the agent "thinks" about what information it is missing and chooses the appropriate tool to find it.

## 🛠️ Tech Stack & Tools
- **Framework:** [LangChain](https://www.langchain.com/)
- **LLM:** OpenAI GPT-4 / GPT-3.5
- **Search Tools:** - `DuckDuckGoSearch`: For real-time web data and current trends.
  - `WikipediaQueryRun`: For historical context and factual grounding.
- **Environment:** Developed in Google Colab.

## 📂 Repository Structure
- `Assignment2.ipynb`: The main source code and execution logic.
- `requirements.txt`: List of Python dependencies.
- `outputs/`: Folder containing sample reports generated by the agent.
  - `sample_1.md`: Impact of AI in Healthcare.
  - `sample_2.md`: AI in Finance & Algorithmic Trading.
